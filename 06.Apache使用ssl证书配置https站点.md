
### apache使用ssl证书搭建https网站 

```shell
NameVirtualHost *:443

<VirtualHost *:443>
    DocumentRoot /web/linux
    ServerName 192.168.1.1
    ErrorLog logs/ceshi-error_log
    CustomLog logs/ceshi-access_log common
    SSLEngine on
    SSLProtocol all -SSLv2
    SSLCipherSuite DEFAULT:!EXP:!SSLv2:!DES:!IDEA:!SEED:+3DES
    SSLCertificateFile /etc/httpd/ssl.crt
    SSLCertificateKeyFile /etc/httpd/linux.com.key
    DirectoryIndex index.php          
   <Directory /web/linux>
       Options all
       AllowOverride all
       Order allow,deny
       Allow from all
   </Directory>
</VirtualHost>
```
