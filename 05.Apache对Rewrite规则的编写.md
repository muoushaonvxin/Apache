
Rewrite 规则可以对来访问的主机实现控制

Apache要使用访问Rewrite规则来对访问服务器的请求做处理

```apache
<VirtualHost *:81>
    RewriteEngine On 
    RewriteRule ^(.*) http://www.baidu.com [L]
    DocumentRoot "/web/ceshi"
    <Directory "/web/ceshi">
        Options Indexes MultiViews FollowSymLinks
        AllowOverride All
        Order allow,deny
        Allow from all
    </Directory>
</VirtualHost>
```
